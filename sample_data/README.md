# Sample Data

Raw data is intentionally not included in this portfolio repository.

Reasons:

- raw scraped files and HAR captures can be large and noisy;
- some sources may have redistribution restrictions;
- the purpose of this repo is to demonstrate methodology, not to publish a full betting dataset;
- private research artifacts and generated model bundles are excluded.

The `reports/` directory contains sanitized aggregate metrics and result tables.

If this were converted into a fully reproducible public project, this folder would contain:

- a small anonymized feature matrix;
- split labels (`train`, `valid`, `test`);
- target column;
- model prediction columns;
- calibration-bin examples.
